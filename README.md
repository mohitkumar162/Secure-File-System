🔐 Secure File System (AWS)
A secure and scalable web application that allows users to upload, manage, and download files with strong encryption and access control. Designed using modern cloud and security best practices.

## 🚀 Features
- 🔑 User authentication and authorization (JWT/OAuth2)
- 🔒 File encryption at rest and in transit (AES-256 + HTTPS)
- 📂 File upload, download, and folder management
- 👥 Secure file sharing with Role-Based Access Control (RBAC)
- ♻️ Soft delete option for recovery
- 📊 Admin dashboard with user and file analytics

## ☁️ AWS Architecture
- 🌐 Designed a custom VPC with public subnet
- 🖥️ Deployed Flask application on EC2 instances
- ⚖️ Used Application Load Balancer (ALB) for traffic distribution
- 🔁 Implemented Auto Scaling Group for high availability
- 📈 Configured CloudWatch for monitoring and alerts
- 🗄️ Integrated Amazon S3 for scalable storage

## 🔐 Security Best Practices
- Least privilege access using IAM roles
- Secure communication via HTTPS
- Data encryption using AES-256
- Scalable and fault-tolerant architecture

## 🛠️ Tech Stack
- Backend: Flask (Python)
- Cloud: AWS (EC2, S3, ALB, Auto Scaling, CloudWatch, VPC)
- Security: JWT, AES-256 Encryption
- Frontend: HTML, CSS, JavaScript
